# desc 组件
> 描述列表组件

# 属性
| 属性名     | 说明           | 类型        | 默认值 |
| ---------- | -------------- | ----------- | ------ |
| list       | 描述列表       | DescItem[]  | -      |
| itemWidth  | 单列宽度       | string      | `50%`  |
| labelWidth | label 宽度     | string      | `80px` |
| labelAlign | label 居中方向 | left, right | -      |
| i18n       | 多语言         | boolean     | -      |
| i18nPrefix | 多语言前缀     | string      | -      |
{ .dense }
